# Проект анализа вакансий DevOps с hh.ru

## Описание проекта

Проект представляет собой комплексное ETL-решение для сбора, обработки и визуализации данных о вакансиях в сфере DevOps с портала hh.ru. Реализована автоматизированная система, которая ежедневно собирает актуальные вакансии, обрабатывает их с использованием Apache Spark и сохраняет результаты в PostgreSQL с последующей визуализацией в Grafana.

## Актуальность

Анализ рынка труда в IT-сфере является важной задачей для образовательных учреждений, студентов и профессионалов. Проект позволяет отслеживать динамику спроса на DevOps-специалистов, требования работодателей и уровень зарплат в конкретном регионе.

## Цели проекта

1. **Сбор данных**: Получение актуальных данных о вакансиях DevOps с публичного API hh.ru
2. **Обработка данных**: Преобразование и очистка данных с использованием Apache Spark
3. **Хранение данных**: Организация структурированного хранения в реляционной БД
4. **Визуализация**: Создание интерактивных дашбордов для анализа тенденций
5. **Автоматизация**: Настройка регулярного выполнения пайплайна через GitHub Actions

## Функциональные возможности

### Основные функции:
- **Сбор данных**: Ежедневный сбор вакансий "DevOps" за последние 3 дня для выбранного региона
- **Обработка**: Извлечение и нормализация ключевых полей (зарплата, навыки, локация)
- **Анализ**: Анализ требований к навыкам и уровня зарплат
- **Визуализация**: Интерактивные графики и таблицы в Grafana
- **CRUD операции**: Демонстрация работы с данными через Spark

### Поддерживаемые технологии:
- **Python 3.9+** с использованием PySpark для распределенной обработки
- **PostgreSQL** для хранения структурированных данных
- **Grafana** для визуализации и аналитики
- **Docker** для контейнеризации инфраструктуры
- **GitHub Actions** для оркестрации пайплайнов

## Архитектура решения

```
└─────────────────┘    └─────────────────┘    └────────┬────────┘
                                                        │
                                                ┌───────▼────────┐
                                                │    Grafana     │
                                                │  (визуализация)│
                                                └────────────────┘
```

### Компоненты системы:

1. **HH Collector** (`hh_collector.py`) - сбор данных с API hh.ru
2. **Spark Processor** (`spark_processor.py`) - обработка и нормализация данных
3. **PostgreSQL** - хранение структурированных данных
4. **Grafana Dashboard** - визуализация результатов
5. **CRUD Operations** - демонстрационные операции с данными
6. **GitHub Actions Workflow** - планировщик выполнения

## Ключевые метрики

Проект отслеживает следующие показатели:
- Количество вакансий по дням
- Средняя зарплата по ключевым навыкам (Docker, Kubernetes, AWS, Python, Ansible)
- Распределение вакансий по городам
- Динамика спроса на DevOps-специалистов
- Требования работодателей к навыкам

## Технические особенности

### Обработка данных:
- **Нормализация зарплат**: Приведение к единой валюте (RUB) с учетом курсов
- **Парсинг навыков**: Автоматическое определение ключевых технологий из описаний
- **Очистка данных**: Фильтрация дубликатов и некорректных записей

### Масштабируемость:
- Поддержка распределенной обработки через Spark
- Контейнеризация всех компонентов
- Возможность расширения на другие регионы
